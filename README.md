# tabChangePlugin
It's a plugin about tabChange;

这是自己写的一个简易Tab切换的插件;

查看demo：http://htmlpreview.github.io/?https://github.com/zaoansijia/tabChangePlugin/blob/master/tabChange.html;

调用方法：

$.tabChange({
        'container' : '#tab',//最外层的容器
        'trigger_event_type' : 'mouseover'//确定鼠标事件,默认是click
    });
